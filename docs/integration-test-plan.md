# 機能一覧 & 結合テスト計画

## 機能一覧

### 1. ドキュメント CRUD

| # | 機能 | 説明 |
|---|------|------|
| 1.1 | setDoc | ドキュメントの作成・上書き |
| 1.2 | setDoc (merge) | 既存ドキュメントへの部分マージ |
| 1.3 | addDoc | 自動IDでドキュメント追加 |
| 1.4 | getDoc | ドキュメント取得 |
| 1.5 | updateDoc | ドキュメントの部分更新 |
| 1.6 | deleteDoc | ドキュメント削除 |

### 2. フィールド値センチネル

| # | 機能 | 説明 |
|---|------|------|
| 2.1 | serverTimestamp() | サーバー側タイムスタンプ自動設定 |
| 2.2 | increment(n) | 数値のアトミックインクリメント |
| 2.3 | arrayUnion(elements) | 配列への要素追加（重複なし） |
| 2.4 | arrayRemove(elements) | 配列からの要素削除 |
| 2.5 | deleteField() | 特定フィールドの削除 |

### 3. データ型

| # | 機能 | 説明 |
|---|------|------|
| 3.1 | プリミティブ型 | string, number, boolean, null |
| 3.2 | Timestamp | 日時型 |
| 3.3 | GeoPoint | 緯度経度型 |
| 3.4 | Bytes | バイナリ型 |
| 3.5 | Map (ネスト) | ネストされたオブジェクト |
| 3.6 | Array (ネスト) | ネストされた配列 |
| 3.7 | DocumentReference | ドキュメント参照型 |

### 4. クエリ — フィルタ

| # | 機能 | 説明 |
|---|------|------|
| 4.1 | == | 等価フィルタ |
| 4.2 | != | 不等価フィルタ |
| 4.3 | <, <=, >, >= | 比較フィルタ |
| 4.4 | array-contains | 配列にある値が含まれるか |
| 4.5 | array-contains-any | 配列にいずれかの値が含まれるか |
| 4.6 | in | 値がリストに含まれるか |
| 4.7 | not-in | 値がリストに含まれないか |
| 4.8 | and() 複合フィルタ | AND条件の組み合わせ |
| 4.9 | or() 複合フィルタ | OR条件の組み合わせ |

### 5. クエリ — ソート・ページネーション

| # | 機能 | 説明 |
|---|------|------|
| 5.1 | orderBy (asc/desc) | フィールドによるソート |
| 5.2 | limit(n) | 結果数の上限 |
| 5.3 | limitToLast(n) | 末尾N件の取得 |
| 5.4 | startAt / startAfter | カーソルページネーション（開始） |
| 5.5 | endAt / endBefore | カーソルページネーション（終了） |
| 5.6 | 複数orderBy | 複数フィールドによるソート |

### 6. コレクショングループクエリ

| # | 機能 | 説明 |
|---|------|------|
| 6.1 | collectionGroup | 同名サブコレクションを横断検索 |

### 7. 集計クエリ

| # | 機能 | 説明 |
|---|------|------|
| 7.1 | count() | ドキュメント数のカウント |
| 7.2 | sum(field) | 数値フィールドの合計 |

### 8. バッチ書き込み

### 9. トランザクション

### 10. リアルタイムリスナー
